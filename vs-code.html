Resumo da Aula de Desenvolvimento WEB

Na aula de desenvolvimento web, aprendi sobre HTML, que é a base de qualquer site. Ele serve para estruturar o conteúdo, como textos, imagens e links. Observei e aprendi como usar tags como < html >, < head >, < body >, < h1 > e < p > para organizar a página. Também aprendi como criar listas, tabelas e adicionar links. Foi muito interessante ver como o HTML é simples, mas super importante para começar a construir um site.
